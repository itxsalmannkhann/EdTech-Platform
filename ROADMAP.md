# Roadmap

This roadmap reflects the planned direction of EduAI. Priorities may shift based
on community feedback.

## ✅ v1.0 — Core platform (shipped)
- Authentication & RBAC (student / instructor / admin)
- Course catalog, detail pages, reviews
- Student / instructor / admin dashboards
- Responsive dark-blue design system
- Prisma schema + seed

## ✅ v1.2 — AI features (shipped)
- AI Tutor with streaming responses
- AI course recommendations
- Personalized dashboard
- Documentation & accessibility improvements
